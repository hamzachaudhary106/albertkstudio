# Albert K Studio — Curly-Inspired Homepage Mockup

A premium hair salon homepage for **Albert K Studio** (Aventura, FL), redesigned to match the layout, typography, and editorial aesthetic of the [Curly Hair Salon template](https://curly.qodeinteractive.com/elementor/hair-salon/) by Qode Interactive.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Design Reference

- **Template:** [Curly — Hair Salon](https://curly.qodeinteractive.com/elementor/hair-salon/)
- **Typography:** Playfair Display (headings) + Lato (body)
- **Palette:** Black & white editorial with subtle warm gray accents

## Page Sections (Curly Layout)

1. Top hours bar
2. Fixed header with centered logo + side menu
3. Scrolling marquee ticker
4. Full-screen hero — "Hair lovers"
5. Our story + 3 featured services
6. Working hours + calendar
7. Transformations gallery
8. Insights / blog cards
9. Meet the stylists
10. Client testimonials
11. Book appointment CTA
12. Perfect hair care split section
13. Footer — Contact, Info, Maps

## Tech Stack

React 19 · TypeScript · Vite · Tailwind CSS v4
